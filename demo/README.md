# Demo — „Raport zarządczy jednym kliknięciem" (Klarow)

Statyczna prezentacja modułu **M2** w brandzie **KLAROW** (kit `ui-kit/skills/company-ui`,
paleta polerowanej stali, wordmark tekstowy). Dane fikcyjne. Zero backendu — to jest
jednocześnie pokaz tezy „narzędzie działa lokalnie, bez serwera".

## Uruchomienie

```
python -m http.server 8765 --directory demo    # z korzenia repo
```

→ http://localhost:8765 (motyw ciemny/jasny przełącznik w topbarze).

`static/css/app.css` i `static/fonts/` to **kopie 1:1 z kitu** (model propagacji z SKILL.md —
po zmianie kitu nadpisz kopie).
