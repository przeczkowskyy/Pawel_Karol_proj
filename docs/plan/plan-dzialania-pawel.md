# Plan działania — Paweł (dziś, 2026-07-22)

> Jedno zdanie na punkt. Rola: marka (logo, wizytówki), LinkedIn, lista firm, skrypt lead-gen, start sprzedaży.

## Razem z Karolem (15–30 min na start)
- [ ] Ustalcie, kto trzyma JDG, i podpiszcie jednostronicowe porozumienie 50/50 (wkład każdego, wspólny przychód, konwersja do sp. z o.o. przy pierwszym płatnym kliencie).
- [ ] Wybierzcie pierwszy moduł DEMO — rekomendacja: **Raport zarządczy (M2)**, bo jest najbardziej efektowny i w pełni read-only.

## Paweł — dziś
- [ ] Dokończ logo Klarow i wizytówki, z **QR do klarow.com** i hakiem „Przyślij nam swój najgorszy Excel".
- [ ] Załóż arkusz listy firm-celów (produkcja / budownictwo / dystrybucja, 50–150 osób) z kolumnami: firma, decydent, champion, trigger.
- [ ] Uzupełnij listę do 100 firm z decydentami (właściciel / CFO) i championami (kontroling / gł. księgowa).
- [ ] Przerób profile LinkedIn obojga pod Klarow (nagłówek = hook „wdrożenie w dni, nie miesiące; dane zostają u Ciebie" + dla kogo).
- [ ] Napisz draft wzoru umowy sprintu (zamrożenie zakresu, płatność 50/50, kod i źródła u klienta, licencja wieczysta) do przekazania radcy.
- [ ] Zbuduj w Claude Code skrypt lead-gen, który monitoruje ogłoszenia „kontroler/analityk ze znajomością Excela/VBA" (sygnał: ktoś traci człowieka-Excela = kupuje teraz) i dociąga firmy do listy.
- [ ] Wyślij dziś pierwsze 20 wiadomości outreach (po 10 na osobę) — to KPI, który realnie decyduje o przeżyciu firmy.
- [ ] Umów 1h u radcy prawnego na wzór umowy sprintu i porozumienie wspólników.

## Zasada dnia
- [ ] Nie ścigaj MALINSKI na szerokość — wygrywamy niszą (twarde liczby produkcji/budownictwa), determinizmem, prawdziwym zero-chmury i tym, że klient gada z twórcą.
